Create and manage a structured task list to track progress on complex tasks.

## When to Use
- Tasks requiring 3+ distinct steps or careful planning
- User explicitly requests a todo list
- User provides multiple tasks (numbered or comma-separated)
- After receiving new instructions: immediately capture as todos
- After completing a task: mark complete, add follow-ups

## When NOT to Use
- Single, straightforward task
- Task is trivial or purely conversational/informational

## Task States
- pending: not yet started
- in_progress: currently working (limit to ONE at a time)
- completed: finished successfully
- cancelled: no longer needed

## Rules
- Update status in real-time; mark completed IMMEDIATELY (don't batch)
- Only ONE in_progress at any time
- Break complex tasks into specific, actionable steps
- Cancel tasks that become irrelevant
